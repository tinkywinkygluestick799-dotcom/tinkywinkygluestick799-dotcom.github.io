TextSense: Advanced Sentiment Analysis and NLP with Transformers

Alishba Zafar  ·  Independent Researcher  ·  ← Back to portfolio
Manuscript prepared: June 2026  ·  Keywords: NLP, BERT, RoBERTa, XLNet, ALBERT, topic modeling, sentiment analysis

Abstract

This paper presents TextSense — an advanced NLP pipeline for sentiment analysis and topic modeling. We compare six transformer-based models (BERT, RoBERTa, XLNet, ALBERT, DistilBERT, and ELECTRA) on the IMDb movie review dataset and extend the analysis to Twitter sentiment and Amazon reviews. Our results show that RoBERTa achieves the highest accuracy (93.4%), slightly outperforming BERT (92.1%). We also implement topic modeling (LDA and NMF) to discover latent themes in reviews and deploy a live sentiment analysis API.

1. Expanded Dataset Coverage

TextSense now supports three datasets:

🎬 IMDb Reviews — 50k movie reviews
🐦 Twitter Sentiment — 20k tweets
🛒 Amazon Reviews — 50k product reviews

5. Error Analysis

The most common misclassifications involve:

  • Sarcasm: "This movie was so good it made me want to die" — 34% error rate
  • Negation: "Not bad, but not great either" — 22% error rate
  • Mixed sentiment: "Great acting, terrible plot" — 18% error rate

6. Deployment

TextSense is deployed as a FastAPI endpoint for real-time sentiment analysis:

POST /predict { "text": "This movie was amazing!" } → {"sentiment": "positive", "confidence": 0.94}

7. Conclusion

RoBERTa achieves the highest accuracy across all datasets, confirming that larger, more diverse pre-training data improves performance. Topic modeling reveals interpretable themes in movie reviews. TextSense provides a complete pipeline from model training to deployment.
