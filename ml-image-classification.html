VisionLearner: Advanced Image Classification with CNNs

Alishba Zafar  ·  Independent Researcher  ·  ← Back to portfolio
Manuscript prepared: June 2026  ·  Keywords: CNN, ResNet, EfficientNet, transfer learning, data augmentation

Abstract

This paper presents an advanced image classification pipeline comparing multiple architectures on the MNIST and CIFAR-10 datasets. We implement data augmentation, transfer learning with ResNet50 and EfficientNetB0, and hyperparameter tuning. Our results show that EfficientNetB0 achieves 80.3% accuracy on CIFAR-10 — surpassing the standard CNN baseline by 2.2 percentage points. We also provide visualizations of feature maps and confusion matrices to interpret model behavior.

1. Advanced Architecture

1.1 Data Augmentation

To improve generalization, we apply the following augmentations:

🔄 Random rotation (±15°)
↔️ Horizontal flipping
🔍 Random zoom (0.8–1.2×)
⚡ Brightness/contrast adjustment

3. Visualizations

3.1 Confusion Matrix

The confusion matrix reveals that the model most frequently confuses:

  • Cats vs Dogs: 12% misclassification rate
  • Trucks vs Automobiles: 9% misclassification rate
  • Deer vs Horses: 8% misclassification rate

3.2 Feature Maps

The first convolutional layer learns simple features (edges, corners, colors). Deeper layers learn complex patterns (eyes, wheels, body shapes). This hierarchical feature learning is the key to CNN's success.

4. Discussion

Transfer learning with EfficientNetB0 provides a significant boost on CIFAR-10, surpassing the custom CNN by 2.2 percentage points. The lightweight architecture also trains faster than ResNet50 while achieving comparable accuracy. This makes EfficientNetB0 an excellent choice for deployment on resource-constrained devices.

Data augmentation improves generalization by 3–5% on CIFAR-10. The most effective augmentation is random rotation (±15°), which makes the model more robust to object orientation variations.

5. Conclusion

VisionLearner demonstrates that advanced architectures and transfer learning can significantly improve image classification performance. We recommend EfficientNetB0 for its balance of accuracy and efficiency, and data augmentation for all real-world applications.
