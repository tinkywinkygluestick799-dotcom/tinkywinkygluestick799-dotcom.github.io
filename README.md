## 👩‍🔬 About Me

I'm a **14‑year‑old AI/ML researcher** from Karachi, Pakistan. I taught myself Python, deep learning, and NLP using free online resources — from YouTube lectures to official documentation. My first neural network ran on a laptop, not a supercomputer.

I've built independent research projects in **computer vision**, **natural language processing**, and **explainable AI**. My goal is to build models that are not only accurate but also interpretable and trustworthy.
