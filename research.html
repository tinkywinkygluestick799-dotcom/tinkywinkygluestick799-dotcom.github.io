Quantum Error Correction and the Mathematics of Stabilizer Codes

This research investigates the error-correction performance of Shor’s 9-qubit code and Steane’s 7-qubit code under depolarizing noise. The project combines theoretical stabilizer code analysis with computational simulations implemented in Python 3.10.

Abstract

Quantum computation offers significant advantages over classical computation but is highly susceptible to noise and decoherence. Quantum error correction provides a framework for protecting quantum information by encoding logical qubits into entangled states of multiple physical qubits. This research analyzes the mathematical structure of stabilizer codes and compares the performance of Shor’s nine-qubit code and Steane’s seven-qubit code under a depolarizing noise model. Both codes are capable of correcting arbitrary single-qubit errors; however, differences in efficiency and structure influence their practical performance.
