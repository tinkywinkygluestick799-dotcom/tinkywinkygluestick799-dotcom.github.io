Quantum Error Correction and the Mathematics of Stabilizer Codes

This research investigates the error-correction performance of Shor’s 9-qubit code and Steane’s 7-qubit code under depolarizing noise. The project combines theoretical stabilizer code analysis with computational simulations implemented in Python.

Abstract

Quantum computation offers significant advantages over classical systems but is highly vulnerable to noise and decoherence. Quantum error correction addresses this challenge by encoding logical qubits into entangled states of multiple physical qubits. This research explores the mathematical structure of stabilizer codes and compares the performance of Shor’s nine-qubit code and Steane’s seven-qubit code under a depolarizing noise model. Differences in redundancy, efficiency, and structure influence their respective logical error rates.
