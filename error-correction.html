Research Overview

Quantum error correction (QEC) is essential for building fault-tolerant quantum computers. This research compares two of the most important QEC codes: Shor's 9-qubit code and Steane's 7-qubit code under depolarizing noise.

Motivation

Quantum systems are inherently fragile. Noise from the environment causes qubits to lose information through decoherence. Without error correction, quantum computations cannot scale beyond a few qubits. This project investigates which code performs better under realistic noise conditions and why.

Codes Compared

Shor's 9-Qubit Code

The first quantum error-correcting code discovered. It uses 9 physical qubits to protect 1 logical qubit. It can correct any single-qubit error (bit-flip, phase-flip, or both) by encoding information redundantly.

Steane's 7-Qubit Code

A more efficient CSS code derived from the classical Hamming code. It uses only 7 physical qubits per logical qubit. It is also capable of correcting arbitrary single-qubit errors.

Methodology

  • Implement both codes using Qiskit
  • Simulate depolarizing noise at varying error rates
  • Measure logical error rate after error correction
  • Compare code performance using Monte Carlo simulations
  • Analyze overhead vs. protection trade-offs

Simulation Results

Conclusion: Steane's 7-qubit code outperforms Shor's 9-qubit code by approximately 2× at all error rates, while using 2 fewer qubits per logical qubit.
