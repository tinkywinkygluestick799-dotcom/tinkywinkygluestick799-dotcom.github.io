QubitShield: Shor vs. Steane Quantum Error Correction

Alishba Zafar  |  Independent Researcher  |  ← back to portfolio
Simulation comparing Shor's 9-qubit and Steane's 7-qubit codes under depolarizing noise.

The Problem

Quantum computers lose information to noise. Which error-correcting code protects it best?

The Approach

We simulate both codes using Qiskit Aer under depolarizing noise (0.1% – 2.0%) and measure logical error rates.

Key Result

Steane's 7-qubit code consistently achieves ≈2× lower logical error rates than Shor's 9-qubit code while using fewer qubits.
