Project Overview

This project uses SHAP (SHapley Additive exPlanations) and LIME (Local Interpretable Model-agnostic Explanations) to explain predictions made by black-box machine learning models, making AI more transparent and trustworthy.

Key Findings

  • SHAP provides globally consistent feature importance rankings
  • LIME is better for quick, local explanations of individual predictions
  • Both tools reveal bias and unexpected model behavior
