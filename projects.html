Project Overview

This project implements fundamental quantum algorithms using IBM's Qiskit framework.

Algorithms Implemented:

  • Deutsch-Jozsa Algorithm — Exponential speedup for function classification
  • Grover's Search Algorithm — Quadratic speedup for unstructured search
  • Quantum Teleportation — State transfer using entanglement
